AWDS - git

The AWDS developers have abandoned its Subversion repository in favor of git.

Git is a distributed revision control system. Every developer has its own copy of the software repository. This means, there is no central server where you can download the current development version. Instead, you can only get the current version of a specific developer.

Known git repositories

The following git repositories are known. You can download the copy of a repository with git clone <address>. To make it easier for to copy-n-paste, each line contains the command for downloading the repository.

As Andre Herms is currently the main developer, his repository is the main one. The code is stable most of the time and bugfixes go directly in this tree. 

git clone http://aland.cs.uni-magdeburg.de/~aherms/awds-aherms.git

Georgs tree contains more experimental and upcoming features. 

git clone http://aland.cs.uni-magdeburg.de/~glukas/awds-glukas.git/
