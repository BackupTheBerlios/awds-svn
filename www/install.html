AWDS - Installation

Installation for Debian Linux

We provide Debian packages.

Add the following lines to /etc/apt/sources.list
deb http://aland.cs.uni-magdeburg.de/debian/ unstable main
deb-src http://aland.cs.uni-magdeburg.de/debian/ unstable main
and run as root
# apt-get update
# apt-get install awds
 That's all.

Installation from scratch

Installing GEA

AWDS requires the GEA library.

Dowload GEA
unpack GEAtar xzf libgea-*.tar.gz
run configure script cd gea-*
./configure
compile GEAmake
become rootsu
... enter password
or
sudo su
install GEAmake install

This will install GEA in the path /usr/local. Alternative destinations can be defined with the configure script: ./configure --prefix=/path-to-destination. If you do so, please notice that you also have to use a different path for the AWDS configuration.

Installing AWDS

Download AWDS
unpack AWDStar xzf awds-*.tar.gz
run configure script cd awds-*
./configure --with-gea=/usr/local/ --sysconfdir=/etc
compile AWDSmake
become rootsu
... enter password
or
sudo su
install AWDSmake install

After this step, AWDS should be installed. The routing daemon is started with start_awds. You have to be root as it requires special privileges. When the routing is correctly running, it will create a virtual network interface awds0. You should check if it was created with ifconfig.

After the basic installation you should continue with the configuration.
