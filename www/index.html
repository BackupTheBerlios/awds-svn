What is AWDS about?

AWDS (Ad-hoc Wireless Distribution Service) is a Layer 2 routing protocol for wireless mesh networks. It provides transparent Ethernet-like access to all participating nodes, thus easily allowing the employment of different higher level protocols like IP (with DHCP), IPv6, AppleTalk, ...
A Linux implementation of AWDS can be downloaded from our download section. Ports to other platforms are highly appreciated!

AWDS 6.0 release candidate

A new major release of AWDS is on the way. It will include changes in the protocol header, which makes it incompatible with prior releases. This was necessary for including a great new feature: route tracing. It allows to track the route a packet takes. The feature will be included in the ping command. The -p option will allow ping to not only check if a node is reachable, but also make it show the path the packets take.
