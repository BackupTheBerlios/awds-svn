2. - 4. March 2007: AWDS at the Chemnitzer Linuxtage 2007.

AWDS was presented at the Chemnitzer Linuxtage 2007. It was presented in the live section together with the OpenWRT project. For those who understand German, the corresponding poster can be downloaded. Furthermore, on Sunday a presentation was done. The German slides are also available for download. An extended abstract of the talk (in English) can be found here.

AWDS 6.0 release candidate

A new major release of AWDS is on the way. It will include changes in the protocol header, which make it incompatible with prior releases. This was necessary for including a great new feature: route tracing. It allows to track the route a packet takes. The feature will be included in the ping command. The -p option will allow ping to not only check if a node is reachable, but also make it show the path the packets take.
