AWDS - FAQ

Q: I get "start_awds: command not found". Why?

A: start_awds is installed (by default) in /usr/local/sbin. Make sure that it is included in the path, e.g. by typing echo $PATH.

Q: I get " ... Permission denied ..."

A: AWDS requires root privileges. Please make sure that you start it as root.
